# 🚗 Checklist TO 11


Sistema desenvolvido para auxiliar equipes operacionais no registro de choque com objetos na faixa de rolamento  em rodovias.

O projeto foi desenvolvido em **HTML5**, **CSS3** e **JavaScript**, utilizando a tecnologia **Progressive Web App (PWA)**, permitindo instalação como aplicativo e funcionamento totalmente offline.

---

# 📋 Objetivo

O **Checklist de Acidentes** nasceu da necessidade de simplificar e padronizar o registro de ocorrências em rodovias.

Seu principal objetivo é proporcionar rapidez, organização e confiabilidade no preenchimento das informações durante o atendimento em campo, mesmo em locais sem conexão com a internet.

---

# ✨ Principais recursos

- ✅ Funciona totalmente offline
- ✅ Instalável como aplicativo (PWA)
- ✅ Compatível com computadores e celulares
- ✅ Interface responsiva
- ✅ Geração automática de relatório em PDF
- ✅ Armazenamento local durante o preenchimento
- ✅ Publicado via GitHub Pages

---

# 📱 Instalação

## Computador

Acesse:

```
https://donzamb-prog.github.io/checklist-acidentes/
```

O navegador permitirá instalar o sistema como um aplicativo.

---

## Android

1. Abra o link acima no Google Chrome.
2. Toque no menu (⋮).
3. Escolha **Adicionar à tela inicial** ou **Instalar aplicativo**.
4. Utilize normalmente, inclusive sem internet.
